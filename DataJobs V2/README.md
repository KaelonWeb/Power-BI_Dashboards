# Data Jobs Dashboard v2.0 w/ Power BI

![Dashboard Page 1](/images/Data%20Dashboard%20V2.0.png)

## Introduction

This v2.0 dashboard was created for **Job Seekers, Job Transitioners and Job Swappers** to solve a common Problem: Information about the data job market is scattered and hard to grasp. It is using 2024 & 2025 data science job postings (including titles, salaries and locations) all built on a single-page interface.  This project was built while going through an online course to learn Power BI.

## Skills

* 🖌️**Dashboard Layout and Design :**  Crafting an intuitive and appealing report layout.
* 🪫 **Power Query (ETL & Data Shaping) :** Performing data cleaning, shaping and transformation.
* 🏠 **Data Modeling :** Building efficient data models with relationships (Star Schema Principles)
* 📊 **Visualizations :**
    * **Core Charts :** Column, Bar, Line.
    * **Cards :** To highlight key performance indicators.
* ⬇️ **Interactive Features :**
    * **Slicers :** Enabling dynamic, user drive data filtering.
    * **Buttons & Bookmarks :** Streamlining how to perceive the data in different methods, including drill-throughs.


## Dashboard Overview

![Dashboard Page 1](/images/Data%20Dashboard%20V2.0.png)

This is your page for all related to the data of the job market. It showcases key KPI's like total job count, median salaries and top job titles to give you a quick understanding of whats happening in the job market all on one page. 

## Conclusion

This is an updated dashboard of the previous project and this showcases Power BI's ability to transform extensive job postings data into a powerful, streamline tool for career analysis. It allows to filter, explore market insights on a single plage, helping to make informed decisions about any career moves.
